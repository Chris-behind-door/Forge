🔌 API 接入指南

让工程设计工作台连接大模型 — 从零开始，一步一步来

概览与推荐

工程设计工作台需要接入大语言模型（LLM）API 才能使用。接入过程很简单：选一个平台 → 注册账号 → 获取 API Key → 在应用设置里填入，总共不到 5 分钟。

以下按成本从低到高排列了所有可用方案：

方案 类型 费用 适合场景
🟢 智谱 GLM-4.7-Flash 免费 0 元（200K 上下文） 日常查询，免费最强
🟢 ModelScope 免费 0 元（有限额） 轻度使用、试用体验
🟣 Ollama 本地 本地 0 元（需电脑配置） 离线使用、隐私要求高
🟠 智谱 / DeepSeek / 通义 按量 几元 ~ 几十元/月 正式使用
🩷 各家 Coding / Token Plan 套餐 ￥29 起/月 重度使用，详见下文对比
💡 如果你不确定选哪个：
  • 先试免费 → 注册智谱，用 GLM-4.7-Flash（免费且 200K 上下文），零成本体验完整功能
  • 觉得好用 → 考虑订阅套餐，重度使用更划算
  • 只是偶尔用 → 按量计费，一个月也就几块钱
  • 断网也要用 → Ollama 本地部署，完全离线

白嫖方案

🟢

智谱 GLM — 免费额度

免费

智谱新用户注册送一定免费 token 额度，GLM-4.7-Flash 模型完全免费。

优势

  • GLM-4.7-Flash 永久免费，200K 上下文窗口
  • 国产模型，中文理解好
  • 本应用内置支持，配置最简单

限制

  • 免费模型能力有限，复杂工程问题不如付费模型（GLM-5）
  • 新用户送的 token 用完后只能用免费模型

接入步骤

  1. 访问 智谱开放平台，注册账号
  2. 完成实名认证（按平台要求操作）
  3. 在「控制台 - API Key」页面点击「创建新的 API Key」
  4. 复制 API Key
  5. 在本应用设置中：
    • Provider 选择 智谱 GLM
    • API Key 填入刚才复制的 Key
    • 模型填 glm-4.7-flash（免费模型）

ModelScope 免费 API

免费

阿里达摩院魔搭社区提供的免费模型推理服务，支持 Qwen、DeepSeek 等主流模型。

优势

  • 完全免费，无需绑定支付方式
  • 支持 OpenAI 兼容接口，接入简单
  • 模型选择丰富（Qwen、DeepSeek、GLM 等）

限制

  • 有调用频率限制（QPM）和每日 token 上限
  • 高峰期可能响应较慢或排队
  • 部分高级模型不可用

接入步骤

  1. 访问 ModelScope 魔搭社区，注册账号
  2. 绑定阿里云账号（在账号设置中关联，这一步必须做，否则无法使用 API）
  3. 完成实名认证（按平台要求上传身份证等）
  4. 获取 API Token：访问 Token 管理页面，点击「创建新的 Token」，复制保存
  5. 在本应用设置中：
    • Provider 选择 自定义
    • Base URL 填 https://api-inference.modelscope.cn/v1
    • API Key 填入刚才复制的 Token
    • 模型填你想用的模型名（如 Qwen/Qwen3.5-35B-A3B）
提示：Token 页面位置比较隐蔽，直接访问 modelscope.cn/my/myaccesstoken 即可。必须先绑定阿里云账号并完成实名认证才能正常调用 API。详细文档参考 ModelScope API 推理文档。

Ollama 本地运行

进阶
⚠️ 进阶方案：Ollama 需要一定的电脑硬件知识，涉及显存、量化、模型选择等概念。如果你只是想快速用起来，建议先选免费或按量计费方案。Ollama 适合有隐私需求、离线场景，或者对 AI 技术本身感兴趣的用户。

Ollama 让你在自己的电脑上运行大模型，完全离线，无需联网，无 API 费用。模型下载后就是一个本地程序，响应速度取决于你的硬件。

你需要了解的概念

  • 参数量（B）：模型的大小指标。7B = 70 亿参数，32B = 320 亿参数。参数越多越聪明，但也越吃资源。
  • 量化（Quantization）：Ollama 默认使用 Q4 量化（4-bit），也就是把模型「压缩」到原大小的约 1/4。效果损失很小，但大幅降低内存需求。你也可以手动选择 Q8（更高精度）或更激进的量化。
  • 显存（VRAM）vs 内存（RAM）：如果有独立显卡（NVIDIA/AMD），模型会优先用显存运行，速度快很多。没有独显则用系统内存（CPU 推理），速度慢但也能跑。
  • 下载大小：Q4 量化的 7B 模型约 4~5 GB，14B 约 8~9 GB，32B 约 19~20 GB。下载后永久占用磁盘空间。

接入步骤

  1. 下载 Ollama：https://ollama.com/download
  2. 安装后打开终端，拉取模型（以 qwen3.5:9b 为例）： 
    ollama pull qwen3.5:9b

    首次拉取需要下载几 GB，请确保网络通畅。换模型只需改名字，如 ollama pull gemma4:26b。

  3. 验证模型已就绪： 
    ollama list

    应能看到刚才拉取的模型和大小。

  4. 在本应用设置中：
    • Provider 选择 Ollama (本地)
    • API Key 留空
    • 模型填你拉取的模型名（如 qwen3.5:9b）
注意：Ollama 需要在后台运行。确保使用应用前 Ollama 已启动。默认监听 localhost:11434，如果改了端口需要在 Base URL 里同步修改。

应用内配置步骤

无论选择哪种方案，在应用中的配置步骤都一样：

  1. 打开工程设计工作台
  2. 点击左侧边栏的 ⚙️ 设置
  3. 在「LLM 配置」区域：
    • Provider：从下拉列表选择对应的平台
    • API Key：粘贴你的 API Key（会加密存储在系统密钥环中，不会泄露）
    • 模型：填入模型名称（如 glm-4-flash、deepseek-v4-flash 等）
    • Base URL：一般不需要改（已预填默认值），除非使用自定义 Provider
  4. 点击保存
  5. 回到聊天页，发送一条消息测试是否正常
安全说明：你的 API Key 使用系统密钥环（keyring）加密存储，不会以明文形式保存在配置文件中。删除配置时 Key 也会一并清除。

常见问题

❓ 提示「未配置 LLM API Key」怎么办？

说明还没有接入大模型。按上面的步骤选择一个方案，获取 API Key 并填入设置即可。

❓ 提示 403 怎么办？

通常是 API Key 无效或过期。检查：

  • Key 是否复制完整（没有多余空格）
  • 账户是否还有余额/额度
  • 模型名称是否正确

❓ 查询很慢或超时怎么办？

  • 检查网络连接（本地模型 Ollama 不需要网络）
  • 高峰期免费模型可能较慢，可以尝试付费模型
  • 本地模型换一个更小的模型（如 4B → 更快的响应）

❓ 可以同时配置多个 Provider 吗？

目前支持保存多个 Provider 的配置，但同一时间只能激活一个。你可以在设置中切换。

❓ API Key 安全吗？

安全。Key 使用系统密钥环加密存储，不会明文保存。应用不会将你的 Key 发送到任何第三方服务器（只发到你选择的模型 API 服务器）。

❓ 大概每月花费多少？

取决于使用频率。参考：

使用频率推荐方案预估月费
偶尔用（每周几次）免费额度 / 按量0 ~ 3 元
经常用（每天几次）按量计费5 ~ 15 元
重度用（每天几十次+）各家套餐（详见对比）￥29 ~ 200 元/月
离线 / 隐私优先Ollama 本地0 元

❓ 为什么选国产而不是 OpenAI？

  • 国产平台大模型中文理解能力优秀，且合规
  • 价格远低于 OpenAI（GPT-4o 输入 30 元/百万 token）
  • 多家厂商提供免费额度和套餐方案
  • 本应用内置支持，配置更方便
